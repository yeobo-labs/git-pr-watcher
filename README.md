# Git PR Watcher

Mini web application that fetches Github repositories and pull requests

## Stack

- Gatsby v2 + TypeScript
- MobX with decorators
- Github GQL API
- TSLint
- Jest

## Usage

1. Create a `.env` file with your personal access token from GitHub
2. Run `gatsby develop`
